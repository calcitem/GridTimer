# Android Notification Sounds

Place your MP3 notification sound files here (same as in `assets/sounds/`):

Required files:
- bell_01.mp3
- bell_02.mp3
- beep_soft.mp3
- chime.mp3
- ding.mp3
- gentle.mp3

**Important:**
- Filenames must use underscores (not hyphens)
- Files must be referenced without extension in code
- Files should be < 100KB each for best performance
