# Changelog

All notable changes to Grid Timer will be documented in this file.

## [Unreleased]

### Added
- Initial implementation of 3x3 timer grid
- Multi-mode support (save multiple grid configurations)
- Exact alarm scheduling for Android 14+
- Full-screen notification for lock screen alerts
- TTS announcements (English and Chinese)
- Customizable ringtones (6 built-in sounds)
- Timer state persistence and recovery
- Permission onboarding wizard
- Bilingual support (English + Chinese)
- Clean Architecture implementation
- Comprehensive PRD documentation

### Technical
- targetSdk 35 (Android 15) for Play Store compliance
- Hive for local persistence
- Riverpod for state management
- flutter_local_notifications for alarms
- Freezed for immutable entities
